##毕业设计

基于Qt5开发的问题对抗系统（Challenge To You），这是我的2017年毕业设计。

github 地址：https://github.com/BaoLearns/ChallengeToYou

环境：Windows/Linux/OS + Qt5 + Sqlite3

使用步骤：

>下载Qt5

>运行Release下的ChallengeToYou.exe
